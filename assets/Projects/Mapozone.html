Mapozone demonstrates Location Management and Analytics Tool


In 2023, I won this project in my freelance career to build a GIS solution for location management of UK, which would allow users to add up their information / data and find datasets of interests for visualization.


Tech stack used is React, React-Leaflet, Google Locations API, Postgresql, PostGIS, Node.js


This was a freelance project that I won over Upwork as a freelancer and developed the portal from scratch to top & deployed it over AWS servers.
